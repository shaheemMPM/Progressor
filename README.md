# Progressor

## This is A Simple Cross Platform Todo App that keep Track of Progress on Listed Todo Items

### Production State

For using do following steps on terminal

```
1. git clone https://github.com/shaheemMPM/Progressor.git
2. cd Progressor
3. npm install
4. electron .
```
